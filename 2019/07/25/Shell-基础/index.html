Shell 基础

2019-07-25

在linux系统中常常需要用shell脚本进行批处理，当然也可以用python脚本处理，这里我总结一些自己常用到的shell简单操作

文件名、下载链接中的数字变量

由于文件数量较大，一个个处理过于耗时耗力，你可以直接读取当前目录下所有文件，也可以对文件名进行循环操作，由于外部链接无法读取，所以只能进行循环，而往往数字都是固定长度的，比如001、002、003…
这里提供一种变量长度固定化的方法
#!/bin/bash for i in $(seq -f %03g 1 100) do echo $i done
